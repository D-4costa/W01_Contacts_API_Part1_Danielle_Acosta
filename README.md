# W01 Project: Contacts API (Part 1)

## Overview
This project provides a RESTful API for storing and retrieving contact information using MongoDB and Node.js.

## Features Completed (Week 01)
- MongoDB connection established
- GET /contacts (Retrieve all contacts)
- GET /contacts/:id (Retrieve single contact by ID)
- MVC Architecture implemented
- Environment variables secured using .env
- Ready for deployment on Render or Mongo

## Contact Fields
Each contact includes:
- firstName
- lastName
- email
- favoriteColor
- birthday

## Security
- MongoDB credentials stored in .env
- .env and node_modules excluded via .gitignore

## Deployment
Deploy to Render and configure environment variables in the dashboard.
